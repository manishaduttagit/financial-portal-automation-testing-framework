# Financial Portal Automation Testing Framework

A professional UI automation testing framework built from scratch using **Python**, **Selenium WebDriver**, and **Pytest**. This project implements the **Page Object Model (POM)** design pattern to ensure scalable, maintainable, and robust test execution.

## 🚀 Key Features
* **Page Object Model (POM):** Clean separation of test logic and page-specific elements for maximum reusability.
* **Explicit Smart Waits:** Implemented robust synchronization using `WebDriverWait` to eliminate flaky test runs.
* **Automated Test Suite:** Includes comprehensive verification for user authentication paths (successful login and explicit error handling).

## 🛠️ Tech Stack & Tools
* **Language:** Python 3
* **Testing Framework:** Pytest
* **Automation Tool:** Selenium WebDriver
* **IDE:** Visual Studio Code
